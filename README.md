### Aumetar screenlight
aumenta el brillo maximo de la pantalla (eDP Nombre de la pantalla, 1.2 Nivel de brillo)
```
xrandr --output eDP --brightness 1.2
```
nombre de la pantalla, `en este caso es eDP`
